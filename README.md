# egg-boilerplate-mp

微信小程序后端模板

## 安装

**Don't `npm i` this directly**

```bash
$ mkdir tell-demo && cd tell-demo # 初始化项目
$ npm init tell # 下载脚手架
$ npm install # 安装模块
$ npm run dev # 运行项目
$ open http://localhost:7001 # 打开浏览器
```

See [create-tell](https://github.com/unclexiao/create-tell) for more detail.
